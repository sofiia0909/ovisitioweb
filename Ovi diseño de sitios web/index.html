  • Competencias generales del curso:

    • Ovi diseño de sitios web.

    En este ovi encontrara documentación de tipo lecturas, videos y actividades para fortalecer el conocimiento en el uso de Git, Github, html5, css3. Cuenta con cinco lecturas de los diferentes temas expuesto, dos videos explicativos sobre sitios web y la importancia de html5 y css3 a demás podrá contar con seis actividades que pretenden reforzar los conocimientos adquiridos.

    Objetivos del Ovi

    ● El ovi pretende dar a conocer el uso básico de git y github ● Conocer el uso de html5 y su estructura. ● Conocer el uso de css3 y sus funciones ● Fortalecer los conocimientos obtenidos por medio de actividades.

    Objetivos Curso

    ● Que el estudiante se fundamente teóricamente sobre las generalidades básicas del Diseño Web. ● Que el estudiante pueda utilizar de forma correcta los lenguajes HTML5 y CSS3 para aplicarlos en la solución de problemas de construcción de Sitios Web. ● Que el estudiante implemente servicios y sitios web a través de sistemas de control de versiones.
